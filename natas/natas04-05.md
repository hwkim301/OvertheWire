## Level4 -> Level5


The website says that access disallowed and also that we're not logged in. 


You should probably check the cookies of the webpage. 


What exactly is a cookie?


A **[cookie](https://en.wikipedia.org/wiki/HTTP_cookie)** os a small piece of dat created by the web server when a user is browsing a website, they enable web servers to store stateful information. 


You can check the cookie values by right clicking-> Inspect-> Applications.


Then you'll find a Storage section on the left side of your devtools and you can find a `Cookies` tab. 


Cookies consist of a `Name` and a `Value`. 


In this level the `Name` is `loggedin` and the `Value` is `0`. 


Change the value to 1 and refresh the webpage using `ctrl+r` or `Cmd +r` for MacOS. 


After refreshing the webpage you'll be able to see the password for the next level. 


```html 
Access granted. The password for natas6 is 0RoJwHdSKWFTYR5WuiAewauSuNaBXned
```
